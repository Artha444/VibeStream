# 🎧 VibeStream

A modern music streaming UI inspired by Spotify, built with React.js.  
VibeStream lets users browse songs, play music (demo playback), and manage favorite tracks.

---

## ✨ Features

- 🎵 Spotify-inspired UI design
- ▶️ Music play simulation (demo audio playback)
- ❤️ Add / remove favorite songs
- 📂 Song list browsing
- 🔍 Clean and modern responsive layout
- ⚛️ Built with React.js (no backend required)
